CSS

In in blog I will going to explain what is css and some of its concepts.

What is CSS ?

It is defined as Cascading Style Sheet which refers to styling of html tags or elements on web.

Some of Properties

1. Padding

It refers to the space around the text element , can be in px rem etc.
This text has 20px padding.

2. Margin

It refers to the space around the border , It can be top,left,right,bottom.
This container has 0.5rem as margin

3. Text-align

It refers to the position of the text elements. User can adjust to start or center or end etc.
This text is align as center

4. Box-shadow

It refers to shadow effect around a elements, It can be a button , a container etc.
This container has shadow

5. Border-radius

It refers to rounded shape around corners of border, This is a most useful property.
This container has border radius of 0.5rem

6. Hover

This property helps if the cursor hover the element what should be happen. The effect can be change of a color , size , font etc.
This text color will change on cursor hover on it

7. letter-spacing

This property mentions the space between the letters , It can be increased or decreased the space.
This text has letter spacing of 2px

8. display

This properties tells the display of content , like flex , block , flow etc.
This container set as flex

9. flex-direction

This property applicable if display is set as flex , after you can mention where to display it in row or column etc.
This container has direction as column

10. text-decoration

It is used when some decoration needed to add to text, like font,color etc.
This text has strike decoration
